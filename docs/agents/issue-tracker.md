# Issue Tracker — GitHub

This repo uses **GitHub Issues** for work tracking.

## Workflow

- Issues are created via `gh issue create`
- Skills `to-prd`, `to-issues`, `triage`, and `qa` read/write to GitHub Issues
- The `gh` CLI must be authenticated (`gh auth login`)

## Repo

- Remote: `https://github.com/ctekv1/rancer.git`
- Issues URL: https://github.com/ctekv1/rancer/issues

## Notes

- No local `.scratch/` tracker is used
- For GitHub Actions / CI, see `.github/workflows/ci.yml`
